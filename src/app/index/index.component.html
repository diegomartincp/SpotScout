Bienvenidos a SpotScout!

Bienvenido a la versión Beta de SpotScout. Esta aplicación se basa en el análisis de sentimiento en posts de periódicos y redes sociales para ofrecerte de forma visualizar el sentimiento general de un lugar para que encuentres el mejor sitio para vivir.

En esta versión, la funcionalidad se limita al análisis de sentimiento con un texto de entrada.

¿Qué es el Análisis de Sentimiento?

El análisis de sentimiento es determinar las emociones que se esconden detrás de un conjunto de palabras. Consiste en un método automático mediante el que se obtienen datos relevantes del cliente (gustos, valoraciones, actos de consumo).
